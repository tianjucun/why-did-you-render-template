# 贡献指南

感谢您对 why-did-you-render-template 项目的关注！

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请：

1. 检查是否已有相关 issue
2. 如果没有，请创建新的 issue，详细描述问题或建议

### 提交代码

1. Fork 本项目
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

### 代码规范

- 使用 TypeScript
- 遵循 ESLint 规则
- 添加必要的注释
- 更新相关文档

### 添加新示例

如果您想添加新的示例项目：

1. 在 `packages/` 目录下创建新项目
2. 确保项目可以正常运行
3. 更新根目录的 `package.json` 添加新的脚本
4. 更新 `README.md` 添加说明

## 开发流程

1. 安装依赖：`pnpm install`
2. 运行示例：`pnpm dev:react18-vite`
3. 进行开发
4. 提交更改

感谢您的贡献！
